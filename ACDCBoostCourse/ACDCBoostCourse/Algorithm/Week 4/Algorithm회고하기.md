## Algorithm 회고하기
 - **1769**
   - s가 3의 배수인지 판별할 때, s가 문자열 형태라서 조건문에 각 케이스를 논리연산자로 표현했다.  
 ``if ( s == "3" || s == "6" || s == "9") cout << "YES" << endl;``  
    - 이때 stoi함수로 문자열을 int형으로 변환한 뒤 나머지연산을 이용하면  조건문을 간결하게 구현할 수 있음을 알게 되었다.  
``if( stoi(s) % 3 == 0) cout << "YES" << endl;``
